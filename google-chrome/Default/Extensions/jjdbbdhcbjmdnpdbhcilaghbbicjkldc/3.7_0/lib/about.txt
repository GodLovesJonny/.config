This add-on contains the following third-party libraries.

layui:
https://github.com/sentsin/layui/tree/v2.4.5/dist
Tip：Modify layui/lay/modules/form.js checkbox to be compatible with vue.min.js

* echarts.min.js
https://github.com/apache/incubator-echarts/blob/4.1.0-release/dist/echarts.min.js

* jquery.min.js
https://github.com/jquery/jquery/blob/3.3.1/dist/jquery.min.js

* qrcode.js
https://github.com/kazuhikoarase/qrcode-generator/blob/v20170724/js/qrcode.js

* snap.svg.min.js
https://github.com/adobe-webplatform/Snap.svg/blob/v0.5.1/dist/snap.svg-min.js
Tip：Change "this" in one place to "window"

* vue.min.js
https://github.com/vuejs/vue/blob/v2.5.21/dist/vue.min.js

* vue-csp.min.js
https://github.com/vuejs/vue/blob/1.0-csp/dist/vue.min.js

* clipboard.js
https://github.com/zenorocha/clipboard.js/blob/v2.0.4/dist/clipboard.min.js
